Haud QA Backend Exercise

Thank you for trying this test.

The aim of this test is a RESTful API on this domain.

Create Test Specification and execute the test for the following items:

  1. List users

    Path: /user
    Method: GET
    Filter (optional): name
    Response: List of users in JSON format.
    Example: curl -sS https://qa-backend-exercise.herokuapp.com/user?name=dorothy | jq
    [ { "name": "Dorothy Gale" } ]

  2. Add user

    Path: /user
    Method: POST
    Payload (JSON) { name: "MY NAME" }
    Response: Message with the added user.
    Example: curl -sS -X POST --header "Content-Type: application/json" -d '{"name":"MY NAME"}' https://qa-backend-exercise.herokuapp.com/user | jq
    { "message": "Added MY NAME" }

  3. Remove user

    Path: /user
    Method: DELETE
    Response: Message with the remove user.
    Example: curl -sS -X DELETE --header "Content-Type: application/json" -d '{"name":"MY NAME"}' | jq
    { "message": "Removed MY NAME" }
